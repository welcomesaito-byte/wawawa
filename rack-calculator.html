仕組み・計算式

各ラックを上から見ると、前方2本は外側、後方2本は内側に配置されています。

FL ────── FR FL ──── FR┊FL ──── FR │ │ │ │ │ │ │ BL ── BR │ │ BL─BR │ │ BL─BR │ 1台（単独） 2台並べた場合

前方柱が外に張り出すため、隣のラックの前方柱と隣接（B）。
後方柱は内に入り隙間ができるため非隣接（A）のまま。

1列あたりの計算式（n = ラック数）：
A = 2n + 2、B = n − 1
検算: A + 2B = 4n（= 総柱数）
